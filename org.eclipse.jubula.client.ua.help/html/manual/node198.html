3.16 Information on Test Steps

Test Steps are the smallest unit in a test. They correspond to one action on one component.

Instead of creating Test Steps , we recommend using the library of Test Cases 3.11.2. Each Test Case in the library contains one Test Step , with all the parameters already referenced. Using the library makes test creation much easier!
