3.27.1.1 Increasing the Java Heap Space for code coverage

Running a test with a code coverage profiler leads to an increased memory requirement for the ITE . You can increase the heap space for the ITE and also enter a package pattern 3.27.1 to reduce the amount of files considered for code coverage.

Users working with a MySQL Database should also follow the steps from the Installation Manual to increase the maximum allowed packet for the Database .
