5. Constants for lists, combo boxes, trees and menus

Many of the actions use parameters for which you need to enter a ''path'' (e.g. menupaths, a path through a tree etc.) or a ''list'' (e.g. to select multiple items from a list or combo box).

The following are the characters you must use to enter lists and paths:

Path Separator
Hierarchical components (e.g. menus, trees) use slash '/' as a value separator. An example menupath is therefore:
File/Open
An example indexpath could be:
1/2
Value Separator
For components which allow more than one item to be selected or checked at once, use a comma ',' to separate items. For example, to select the values ''blue'', ''green'' and ''orange'' from a list, the parameter would look like this:

blue,green,orange

Do not enter a space between list items.

If you are testing a component whose actions use these separators, and you need to enter a comma or slash as a part of the actual parameter, you will need to mask the comma or slash 8.
