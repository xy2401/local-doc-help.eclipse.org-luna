6.14 Summary

After reading this chapter, the structure and concepts of the tool should be clear. You can specify reusable Test Cases to make Test Suites in a Project . To enhance the reusability and structure of test elements and the ease of use, there are various features supported by the program. To look at these features in more detail and find out how to use them, see the "Tasks" chapter 3.
