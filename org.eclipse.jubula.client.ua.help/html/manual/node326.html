3.29.6.1 Using search results to make wide-reaching changes to your Project

Once you have performed a search, you can perform specific actions on the results:

  • You can replace a Test Case at multiple places where it has been reused with another Test Case 3.11.7.
  • You can change the column of a central test data set used by any Test Cases that use this central test data set 3.12.9.7.
