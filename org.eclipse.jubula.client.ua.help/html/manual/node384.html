3.37.8 My Test Suite failed when using rdesktop

  • Under a very specific configuration involving rdesktop 1.5, tests are known to frequently fail with an action error: 'Timeout received before confirming the posted event'.
  • This occurs when rdesktop 1.5 is used to log into a user account used for the ITE as well as a separate account for the AUT Agent .
  • The problem does not occur under rdesktop 1.4.1.
