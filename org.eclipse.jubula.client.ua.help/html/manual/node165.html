3.12.9.2 Deleting central test data sets

You can delete a central test data set if the data set has not yet been reused (referenced) in a Test Case 3.12.9.5.
  1. Select:
    Delete

    from the context-sensitive menu or press »DELETE«.

  2. A dialog will appear if the data set has been reused and cannot be deleted.
  3. You can use the search 3.29.4 to show where the data set has been used.
