6. Overview of Components

The following diagram shows which abstract components are used and for which actions they can be used for.

Figure 6.1: Component structure

Figure 6.1 provides an overview of the concrete and abstract component structure. This figure can be interpreted as follows: the arrows show inheritance relations between the components. Thus, a Label component has no actions of its own, but inherits them all from Component with Text. The Label component also inherits any actions that Component with Text also inherited. In this case, these are the actions from Graphics component.
