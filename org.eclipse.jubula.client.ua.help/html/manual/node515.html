6.6.5 Projects

Tests are organized into Projects . A Project is the top-level container. Projects contain runnable tests and the details about the AUT's to be tested (how to start them, in which language etc.).

You can work jointly on Projects using the multi-user Database , and you can also share Projects using the import and export functions.
