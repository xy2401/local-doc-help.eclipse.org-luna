4.7.2 Setting up an iOS AUT for testing

If you want to test an iOS application, you have to prepare the AUT in order to make it testable.

This preparation is designed to be undertaken by a developer who has access to the AUT 's source code as well as knowledge of developing for iOS using Objective-C and Xcode. These instructions assume you are using Xcode 4. For Xcode 3 please adapt the instructions accordingly.
