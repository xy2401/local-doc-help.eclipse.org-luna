3.23.1 The Dashboard application

The Dashboard is a web application that allows you to view test results from within a browser. You can only see test results which are still in the Database 3.20.3.1. If you still wish to see test results after a Database migration, then you should ensure that the old Database instance and a Dashboard in the correct version remain installed so that older test results can be read.
