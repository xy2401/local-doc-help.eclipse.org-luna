3.7.2 Basic information required for every AUT configuration

Regardless of the type of AUT you want to start, you will always need to enter the following information in the AUT configuration dialog.

  • A suggested name for this AUT configuration is generated automatically based on your AUT name and the default AUT Agent host. You can change this name if you wish.
  • The default AUT Agent host is also automatically selected. You can select a new AUT Agent from the combo box or add a new one by clicking "New". For more information on adding and editing an AUT Agent , see the section later 3.28.2.
  • Enter the AUT ID that will be given to this AUT when it is started.
