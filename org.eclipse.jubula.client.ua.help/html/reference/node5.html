4. Components, Actions, and Parameters

This section provides a reference of all currently supported AUT components and the actions that can be applied to them.

There are two types of actions that can be applied to a given component: execute and check. Execute actions are marked with an 'e' in the description, whereas check actions are marked with a 'c'.
