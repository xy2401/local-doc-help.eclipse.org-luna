Example Sources

You can download the complete models used in this tutorial from the following link as ZIP file:

  • multi-model.zip

The ZIP file contains the Provider.xpdl and Consumer.xpdl model files that are used in this tutorial. Note that when you download these models, you may need to provide appropriate path for the file connection. Also, you should configure the mail application server to make the model work. While deploying, first deploy the Provider.xpdl and then Consumer.xpdl. The deployment order is necessary to resolve the references.
