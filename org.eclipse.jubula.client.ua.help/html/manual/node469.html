5.2.2 The Test Case Browser

The Test Case Browser is by default in the lower left-hand corner of the Functional Test Specification Perspective . In this browser, you can create Test Cases , which are the ''building blocks'' for your tests. They are used to group Test Steps and other Test Cases .

More information on working with browsers is available in the Tasks chapter 3.8. To make working with larger Projects easier, you can also have the Test Case Browser open more than once 3.8.4.

You can filter in the Test Case Browser using the field at the top. Use star "*" as a wild card.
