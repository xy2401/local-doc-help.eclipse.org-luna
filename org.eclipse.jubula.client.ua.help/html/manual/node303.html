3.28.3 Embedded AUT Agent preferences

You can find the preference page for the embedded AUT Agent under:
Test-AUT Agent -Embedded AUT Agent
in the preferences dialog.

On this page, you can enter the port number that the embedded AUT Agent 3.1.2 should use when you connect to it 3.1.2. The default is 60001.
