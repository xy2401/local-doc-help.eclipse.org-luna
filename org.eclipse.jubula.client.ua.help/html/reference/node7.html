4.1.1 Button Component

  • It is implemented by all components which can be clicked like buttons.

Check Selection (Button Component)

  • Use this action to check if the button is currently selected or not.
  • This action is particularly useful for check boxes and radio buttons.

Parameters
Name Data Type Values Default
Selected Boolean true
false
true

  • Set this parameter to true if you expect the button to be selected.
  • Set this parameter to false if you expect the button to not be selected.
