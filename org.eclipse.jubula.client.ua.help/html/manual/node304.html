3.28.4 Database preferences

You can find the Database connection preference page under:
Test-Database Connections
in the preferences dialog.

When you start the ITE for the first time, it is automatically configured to use an embedded Database . If this is the only Database configured, then the Database login is performed automatically.

In this preference page, you can configure other Database connections for your productive Database .

We do not recommend using the embedded Database for productive use.
