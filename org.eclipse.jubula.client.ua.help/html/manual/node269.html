3.23.5 Connecting to the Dashboard in the browser

Once a Dashboard server has been started 3.23.4, then you can start the web application in the browser.

Enter the following URL: 

http://HOSTNAME:PORTNUMBER/dashboard

Once the application has loaded, you will see the Functional Test Reporting Perspective and can use it to view and analyze test results 3.23.6.
