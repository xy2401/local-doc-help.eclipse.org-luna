4.2.3 Keyboard Layouts

For RCP AUT's , a keyboard layout must be entered in the AUT configuration 3.7.5.3. German (DE) and English (US) are provided as standard keyboard layouts.

If you require a different keyboard layout, you must create a mapping file for the new language -->Reference Manual and place it in the server/resources directory. You also have to put this mapping file into the RCP Remote Control plug-in of your AUT :

<RCPAUT>/plugins/org.eclipse.jubula.rc.rcp_<Version>/resources
