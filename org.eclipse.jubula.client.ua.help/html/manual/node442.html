4.7.3.1 Naming components

To be able to robustly test iOS AUT's , we highly recommend naming the components in your AUT . The name that is used as a part of the object recognition is the accessibility identifier. This is a variable that can be specified for each UI component in an iOS AUT . It is language-independent and is designed for use in automated tests. We recommend using unique names throughout the AUT .
