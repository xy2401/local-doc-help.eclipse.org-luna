3.5.9 Exporting Projects

You can choose to export the currently opened Project 3.5.9.1 or all of the Projects in the database 3.5.9.2. Projects are exported in .xml format so that they can be added to e.g. version control systems.

Projects are exported along with all their test result summaries.
