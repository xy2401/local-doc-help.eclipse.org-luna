4.7.5.7 Working with unmappable (unsupported) components

If there is a component that is unsupported, then you may be able to tap it using the action in the unbound_modules_ios called Tap View with Accessibility Label. For more information on this action, see the section on working with the keyboard 4.7.5.6.
