3.23.2 Starting the Dashboard application in the browser

Once the Dashboard server has been started, you can start the Dashboard application in the browser. This happens automatically if you are using the dashboard application 3.23.3, or you can enter a URL to start the web application and connect to the server you started 3.23.5.
