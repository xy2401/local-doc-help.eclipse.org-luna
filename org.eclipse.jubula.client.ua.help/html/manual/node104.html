3.7.7.1 AUT configuration for Windows desktop AUT's

To be able to start a Windows desktop AUT , you should select the win toolkit as the AUT toolkit in the AUT definition dialog.

You must enter the following information to be able to start a win AUT :

  1. Enter the basic configuration details as described earlier 3.7.2.
  2. Enter the executable file name in the Executable File Name field. This path can be relative if you define a working directory 3.7.3).
  3. Enter any necessary command-line arguments for the AUT in the AUT Arguments field.
