3.27 Working with code coverage with Java tests

The ITE is integrated with the JaCoCo code coverage tool for Java applications to let you run and analyze code coverage for your tests.

Code coverage is only available for AUT's started via an AUT configuration 3.7.1 (i.e. not using the autrun command), and which use Java 1.5 or higher.
