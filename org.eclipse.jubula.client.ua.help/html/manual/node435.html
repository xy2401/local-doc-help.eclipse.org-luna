4.7.1 Supported iOS AUT's

The following AUT's written for iOS are supported:

  • You must be using iOS SDK 5.0 or higher. iOS AUT's on version 6.1 and 7.0 of iOS are supported.
  • You can run tests either on a simulator (i386) or on a real device (armv7 and armv7s).
