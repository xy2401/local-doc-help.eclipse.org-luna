4.5 rcp Toolkit

The RCP toolkit contains components that are only found in SWT applications. You will only have access to these components in your tests if you choose RCP or SWT as the Project toolkit.

Most of the RCP components are covered in the section on SWT. Components in GEF AUT's are described in the section on GEF.
