3.19.2 Starting the AUT

To start the AUT , you must:

  • have defined an AUT 3.6
  • have connected to the AUT Agent 3.1.2
  • if you want to start your AUT from the ITE , and not using the autrun option, then you must have configured an AUT 3.7.1

You can only start AUT's from the ITE which support the current working language. Change the working language if necessary by selecting a language from the list next to the language button on the toolbar.

  1. On the toolbar, click on the arrow next to the "Start AUT " button.
      start AUT

  2. Select an AUT and the configuration to start it with from the list.
  3. The AUT will be started in the current working language.
  4. Clicking directly on the "Start AUT " button will start the most recently started AUT (marked with a tick in the list of AUT's and configurations).

Once the AUT has been started, it will appear in the running AUT's view.
Further information on starting specific types of AUT is available in the toolkit-specific details 4.
