3.11.4 Editing Test Cases

From the Test Case Editor , you can edit a Test Case . You can:

  • add other Test Cases 3.9.2.
  • add and edit Test Steps 3.16.1.
  • rename the Test Case 3.8.1 or the Test Cases it contains 3.9.4.
  • add a comment to the Test Case 3.9.5.
  • add, edit or translate data for the Test Case 3.12.11.
  • extract Test Cases from within this Test Case to make reusable modules 3.9.8.
  • deactivate specific Test Cases in this Test Case so they are ignored during execution 3.9.7.
  • replace one or more Test Cases in it with another Test Case 3.9.9.
