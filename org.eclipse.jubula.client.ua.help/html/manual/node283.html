3.24.3 Using the test executor with the embedded Database

If you are using the default embedded Database , you will need to enter the following information as parameters:

-dbscheme
Default Embedded (H2)
-dbuser
sa
-dbpw
<empty> ("''''")

If you have changed the name of the default Database scheme in the Database preferences 3.28.4, you should alter the parameter for the -dbscheme accordingly.
