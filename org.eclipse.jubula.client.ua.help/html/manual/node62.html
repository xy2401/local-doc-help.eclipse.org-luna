3.5.2.4 Editing the AUT's in a Project

To see and edit your AUT's for a Project , select "AUTs" from the tree on the left-hand side of the Project properties dialog.

In the screen that appears, you can see any AUT's you have already added to the Project (3.5). You can choose to edit them, delete them or add a new AUT .

Figure 3.5: AUT Properties Dialog

Defining and editing AUT's from here involves the same steps as defining an AUT in the Project wizard 3.6.
