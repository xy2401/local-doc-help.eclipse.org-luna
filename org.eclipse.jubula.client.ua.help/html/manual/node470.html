5.2.3 The Component Name Browser  

The Component Name Browser   (5.5) is by default in the lower left-hand corner of the Functional Test Specification Perspective , behind the Test Case Browser . In this browser, you can see component names used in this Project and in other Projects . You can rename, add, delete and merge component names and can also find where you have used component names in your test.

You can filter in the Component Name Browser   using the field at the top. Use star "*" as a wild card.
