3.7.4.7 Creating an AUT definition from a running AUT

Once you have started an AUT using the autrun command, you can automatically generate an AUT definition 3.6 for this AUT :

  • In the Running AUT's View , select the AUT you want to define (it will be marked as an unknown AUT ID).
  • Select:
    Create AUT Definition

    from the context menu.
  • The AUT definition window will appear. Complete the dialog 3.6 and click "OK".
