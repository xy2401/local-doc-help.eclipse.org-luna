3.7.5.2 Basic JavaFX AUT configuration

You can use the basic setting to configure your AUT if it can be started by an executable file (e.g. .bat, .exe, .cmd, .sh etc.). If your AUT is a JAR, then you can also use the executable file field to enter the Java executable (i.e. java or java.exe) and then enter the path to the JAR in the AUT Arguments field in the advanced configuration 3.7.5.3.

You must be using a Java 8 JRE to be able to start JavaFX AUT's .

  1. Enter the basic configuration details as described earlier 3.7.2.
  2. Enter the executable file name in the Executable File Name field. This path can be relative if you define a working directory 3.7.5.3).

For information on the advanced properties for the AUT configuration, see the next section 3.7.5.3.
