4.5 Testing HTML AUT's

HTML AUT's are supported for testing. To be able to test HTML applications, select the HTML toolkit in the Project properties.

When you select HTML as the Project toolkit, the library Projects unbound_modules_concrete and unbound_modules_html are automatically reused in your Project . The actions in these libraries are described in the reference manual -->Reference Manual .
