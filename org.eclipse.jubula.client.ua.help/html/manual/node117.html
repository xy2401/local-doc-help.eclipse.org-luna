3.8.5 Opening the task editor for items in browsers

Each Test Case , Test Suite and Test Job in a Project can be joined to a task in an external repository via a task ID 3.22. You can open the task for an item from the browser:

  1. Select the item whose task you want to open in the browser.
  2. Right click and select:
    Open with --> Mylyn Task Editor
  3. The task from the external repository will open in the ITE .
If no repository has been defined for the Project , or the task ID does not exist, or the repository is not configured in the workspace, then the task will not open.
