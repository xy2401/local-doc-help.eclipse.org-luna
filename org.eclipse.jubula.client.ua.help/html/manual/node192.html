3.15.1 Combining Test Suites into a Test Job

You can create a Test Job when you want to execute a collection of Test Suites after each other.

The Test Suites can test different instances of the same AUT , or different AUT's entirely.

You do not have to create a Test Job if you only want to run tests on one instance of one AUT . To do this, you can use a Test Suite 3.14. A Test Job is most useful when you want to switch between two AUT's (either the same AUT or a different one) in one test run.
