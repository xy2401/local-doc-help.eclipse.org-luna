4.7.2.1 Add the library to your project files

The first step is to link the librc.mobile.ios.nativ static library and its header file directly into your iOS application. This lets your project and thereby your iOS application be run as a testable AUT .

  1. Locate the development/iOS-support.zip file in the installation directory in Finder.
  2. Unzip it and drag all of its content into the Project Navigator.
