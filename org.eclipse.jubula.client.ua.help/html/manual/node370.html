3.36.4 Starting the AUT

The Start AUT button can be used, just like the Run or Debug buttons, to launch an application or configure application launches. Applications started in this way are always started in debug mode. Furthermore, Start AUT adds additional information to the launch process, allowing the started application to be tested.

 Start AUT

Some of the additional information that makes the started application testable can be configured from the Test tab of the Start AUT Configurations dialog.

The AUT ID can be assigned or left blank. If left blank, the configuration name (ex. New_configuration) will be used.
