3.25 Using the dbtool client to import, delete and export from the command line

You can use the dbtool to import, export and delete Projects from the Database in without using the ITE . You can also use it to create a new version of a Project in the Database .

This is particularly useful for automated build and test processes, where you may want to automatically export and import Projects out of and into version control to run your tests.
