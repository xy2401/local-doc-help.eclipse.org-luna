4.2.1 Supported RCP AUT's

AUT's written with the RCP GUI toolkit are supported according to the following points:

  • The AUT is written using Java 1.5 or higher.
  • It uses a Java SE (Standard Edition) JRE.
  • The AUT must use SWT version 3.1 or higher. For Mac systems, the version of SWT used must be 3.6 or higher.
  • Eclipse versions 3.1-3.x and 4.x (with and without the compatibility layer) are currently testable.
