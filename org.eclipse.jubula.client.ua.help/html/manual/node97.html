3.7.5.3 Advanced JavaFX AUT configuration

The advanced configuration dialog lets you create a working directory for your AUT , and add command-line arguments needed to start the AUT . If you entered a Java executable as the executable file name in the basic configuration, then you can enter "-jar <JARFILE>" in the AUT Arguments field (3.15).
